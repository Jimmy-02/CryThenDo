# CryThenDo

CryThenDo is a simple and modern task management web application designed to help users turn pressure into action.  
Instead of overthinking, just cry - then do.

---

## Overview

CryThenDo focuses on simplicity and productivity.  
It allows users to manage daily tasks with a clean interface and smooth experience.

Live demo: https://crythendo.onrender.com/  
Repository: https://github.com/Jimmy-02/CryThenDo  

---

## Features

- Create, update, and delete tasks  
- Organize tasks efficiently  
- Clean and minimal user interface  
- Responsive design for multiple devices  
- Real-time or dynamic task updates  

---
